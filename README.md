## Installation

Adding this repository to your Home Assistant instance is straightforward.

1. Navigate to **Settings → Add-ons → Add-on Store**.
2. Click the **⋮ menu** (top-right) and select **Repositories**.
3. Paste the URL below and click **Add**:
   ```
   https://github.com/Thepowa753/hassio_tunarr
   ```
4. The **Tunarr** add-on will now appear in the store. Click it and press **Install**.
5. Adjust the web port if needed (default: `8000`), then click **Start**.
6. Open the **Web UI** tab or go to `http://<your-ha-ip>:8000`.
